# Flavors of Java

Java comes from coffee and is very much alike. You may find its various flavors, ways of making and additions. You can’t possibly know all of them, but you should know how to explore them and find your favorite. You may also want to know what makes a good programmer or even reach beyond Java.

## Write once, run anywhere

The main Java selling-point that is still valid after 30+ years. Plus backwards compatibility, meaning: write once, run (practically) infinitely.

## Java as simple C

Discover Java as yet another programming language, somewhat similar to C with wothout memory management burden.

## Object Oriented Programming

Java is primarily an OOP language. Model the reality using classes, fields and methods that represent objects and activities taken directly from your business domain.

## Type-level programming

Dive deeper into OOP by modelling the program and data flow by using types. Let types guide the program execution. Often combined with functional programming.

## Functional Programming

If you like math this is for you - a program as a set of _pure_ functions that transform input to output. Embrace immutability that helps in reasoning about the code.

## Backend

Java is a king (or queen...) of backend. By using SpringBoot you are able to provide services to any distributed application you may think. Try not to lose yourself in a sea of libraries.

## Test-Driven Development

Although not specific to Java, it is worth knowing it exists and how useful it may be. At first counterintuitive, when done right it may become your primary way of writing complex programs.

## Beyond Java

Java is not just a programming language, it relies on Java Virtual Machine that opens infinite possibilities. Learn about some other JVM-compatible languages.
